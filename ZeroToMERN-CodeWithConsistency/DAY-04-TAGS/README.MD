# 🌐 HTML Tags and Their Types

## 📌 What are HTML Tags?
HTML tags are used to create elements on a webpage and define the structure of a website.

### ✅ Example
```html
<p>Hello World</p>
```

---

# 📌 Types of HTML Tags

## 🔹 1. Paired / Container Tags
These tags have both opening and closing tags.

### ✅ Example
```html
<h1>Heading</h1>
<p>This is a paragraph</p>
```

---

## 🔹 2. Empty / Self-Closing Tags
These tags do not require a closing tag.

### ✅ Example
```html
<br>
<hr>
<img src="image.jpg">
```

---

# 📌 Structure of HTML

```html
<!DOCTYPE html>
<html>

<head>
    <title>My Website</title>
</head>

<body>
    <h1>Hello World</h1>
    <p>Welcome to HTML</p>
</body>

</html>
```

---

## 📌 Structure Explanation

| Tag | Purpose |
|------|----------|
| `<html>` | Root element of HTML |
| `<head>` | Contains metadata, title, links |
| `<body>` | Contains visible webpage content |

---

# 📌 Semantic Tags

Semantic tags provide meaning to webpage content.

| Tag | Purpose |
|------|----------|
| `<header>` | Top section of webpage |
| `<nav>` | Navigation links |
| `<main>` | Main content |
| `<section>` | Group of related content |
| `<article>` | Independent content/article |
| `<footer>` | Bottom section/footer |

### ✅ Example
```html
<header>Website Header</header>

<nav>Navigation Menu</nav>

<main>
    <article>
        Blog Content
    </article>
</main>

<footer>Footer Area</footer>
```

---

# 📌 Formatting Tags

Formatting tags are used to style text.

| Tag | Purpose |
|------|----------|
| `<b>` | Bold text |
| `<i>` | Italic text |
| `<u>` | Underline text |
| `<mark>` | Highlight text |
| `<strike>` | Strike text |

### ✅ Example
```html
<b>Bold Text</b>

<i>Italic Text</i>

<u>Underline Text</u>

<mark>Highlighted Text</mark>

<strike>Deleted Text</strike>
```

---

# 📌 Common HTML Tags

| Tag | Purpose | Example |
|------|----------|----------|
| `<h1>` | Heading | `<h1>Title</h1>` |
| `<p>` | Paragraph | `<p>Hello</p>` |
| `<a>` | Link | `<a href="#">Link</a>` |
| `<img>` | Image | `<img src="img.jpg">` |
| `<div>` | Container | `<div>Content</div>` |
| `<span>` | Inline container | `<span>Text</span>` |
| `<ul>` | Unordered List | `<ul><li>Item</li></ul>` |
| `<table>` | Table | `<table></table>` |
| `<form>` | Form | `<form></form>` |

---

# 📌 Block-Level and Inline Tags

## 🔹 Block-Level Tags
Take full width of the page.

### ✅ Examples
```html
<div></div>
<p></p>
<h1></h1>
```

---

## 🔹 Inline Tags
Take only required width.

### ✅ Examples
```html
<span></span>
<a></a>
<b></b>
```
