## Topic: Website Fundamentals 🌐  

### 📌 1. What is a Website?
A website is a collection of web pages that are accessed through the internet using a browser.

---

### 📌 2. Types of Websites

#### 🔹 Static Website
- Fixed content (same for all users)
- Built using HTML, CSS
- No backend or database
- Example: Portfolio, simple landing pages

#### 🔹 Dynamic Website
- Content changes based on user interaction
- Uses backend (Node.js, PHP, etc.) and database
- Example: Facebook, YouTube

---

### 📌 3. What is a Web App?
A web application is an interactive website where users can perform actions.

✔ Example:
- Login systems  
- Online shopping  
- Forms & dashboards  

👉 Difference:
- Website → shows information  
- Web App → allows interaction  

---

### 📌 4. How a Website Works

1. User enters URL in browser  
2. Request goes to server  
3. Server processes request  
4. Response (HTML, CSS, JS) is sent back  
5. Browser displays the page  

---

### 📌 5. Structure of HTML

Basic structure of an HTML page:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>This is my website</p>
</body>
</html>


📌 6. HTML Tags

Tags are used to create elements on a webpage.

✔ Common Tags:

<h1> to <h6> → Headings
<p> → Paragraph
<a> → Link
<img> → Image
<div> → Container
<span> → Inline container


📌 7. Sections of HTML

<head> → Contains metadata, title, links
<body> → Visible content
<header> → Top section
<footer> → Bottom section
<section> → Group of content
<article> → Independent content
